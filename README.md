# Configuração

- ter o mysql na máquina
- alterar o .env, nas variáveis

DB_DATABASE=store
DB_USERNAME=user
DB_PASSWORD=passwd

- Gerar os comandos:
- php artisan migrate
- php artisan serve --port 8003

- acessar
http://localhost:8003/
